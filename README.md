# Smart Food Surplus Redistribution System

An IoT-based smart food monitoring and redistribution system developed to reduce food waste and improve surplus food management

---

## Project Overview

The system helps monitor surplus food conditions and supports efficient redistribution between donors and NGOs. It combines IoT hardware, cloud services, and a Flutter mobile application to provide real-time food monitoring and communication.

---

## Features

- Real-time temperature monitoring
- Food weight tracking using load cell sensors
- Firebase cloud database integration
- Mobile application built with Flutter
- GPS-based donor and NGO location tracking
- Sensor-based food monitoring system
- Real-time data communication between devices and application

---

## Technologies Used

### Mobile App
- Flutter
- Dart

### Backend & Cloud
- Firebase
- Firebase Realtime Database

### Hardware & IoT
- ESP32
- DS18B20 Temperature Sensor
- Load Cell
- HX711 Amplifier Module

### Additional Packages
- Geolocator

---

## Hardware Components

| Component | Purpose |
|---|---|
| ESP32 | Main microcontroller |
| DS18B20 | Temperature monitoring |
| Load Cell | Food weight measurement |
| HX711 | Load cell signal amplification |

## Project Goal

The main goal of this project is to reduce food waste by monitoring food conditions in real time and improving surplus food redistribution between food donors and organizations.
